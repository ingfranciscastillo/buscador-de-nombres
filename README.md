# Buscador de Nombres

Encuentra el nombre perfecto para tu mascota, personaje de juego o NPC. Explora cientos de nombres organizados por estilo, género, idioma y más.

## Características

- **Búsqueda avanzada**: Filtra por tipo (mascota/personaje/NPC), estilo (tierno/épico/moderno/fantasía), género, longitud e idioma
- **Favoritos**: Guarda tus nombres preferidos
- **Historial**: Accede a tus búsquedas recientes
- **Nombres populares**: Descubre los nombres más votados
- **Generador aleatorio**: Sorpréndete con un nombre al azar

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Radix UI

## Getting Started

```bash
# Instalar dependencias
npm install

# Iniciar desarrollo
npm run dev

# Build para producción
npm run build
```

## Estructura del proyecto

```
src/
├── components/      # Componentes UI
├── context/         # React Context (favoritos)
├── data/            # Datos de nombres
├── hooks/           # Custom hooks
├── lib/             # Utilidades
└── types/           # TypeScript types
```
